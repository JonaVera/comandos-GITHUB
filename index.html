GITHUB

Prmeros pasos

Estoy iniciando un desarrollo de una app

concetarla a GITHUB

  • Ver estatus
    • git status
  • Ver si ya estamos conectados a una rama
    • git remote -v
  • INICIAMOS GIT
    • git init
  • VAMOS A GITHUB (ARRIBA) Y CREAMOS UN NUEVO REPOSITOR OTRO NOMBRE
    • crear nuevo repositorio
    • agregar nombre nuevo
    • tocar e boton crear
  • conexion remote add
    • Copiamos el comando: git remote add origin https://github.com/JonaVera/comandos-GITHUB.git
    • lo pegamos en nuestra consola y damos ENTER
  • Renombre a rama MASTER -> MAIN
    • Copiamos el comando: git branch -M main
    • lo pegamos en nuestra consola y damos ENTER
  • ARMAR el commit(la cajaa salvar). Nuestra version(nuestro back)
    • git add .
  • COMMIT - Poner nombre a la caja y git lo identifica con un bash
    • git commit -m "mi primera subida de un repo a GIT "
  • PUSH - subir, empujar lo guardado hacia arriba, pero este push por primera y unica vez es un push especial
    • git push -u origin main
    • Para futuros push solo se debe hacer: git push
